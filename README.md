# File Sorter App

自動仕分けシステム - ファイル整理を効率化するデスクトップアプリケーション

## 概要
指定したフォルダ内のファイルを、特定の条件に基づいて自動的に分類・整理を行うElectron製のツールです。手作業でのファイル管理コストを削減し、作業効率を向上させることを目的として開発しました。

## 使用技術
- **Electron**: デスクトップアプリのフレームワーク（クロスプラットフォーム対応）
- **Frontend**: HTML5, CSS3, JavaScript (jQuery)
- **Backend**: Java (バックエンドのロジック処理として `FileSorter.jar` を統合)
- **Tooling**: Git / GitHub

## この制作物のポイント
1. **技術のハイブリッド化**: フロントエンド（Electron）とバックエンド（Java）を連携させ、ロジックとGUIを分離する堅実なアーキテクチャ設計を行いました。
2. **UI/UXへのこだわり**: DJミキサーをモチーフにした独自デザインを採用し、単なるツールを超えた「操作する楽しさ」のあるインターフェースを実現しました。
3. **実用性と解決能力**: 自身の課題であった「ファイル整理の煩雑さ」を、開発スキルを駆使して自ら解決するプロセスを重視しました。

## 開発環境
- macOS
- VS Code


アプリケーションの起動
本プロジェクトのビルドおよび実行は、以下の手順で行ってください。

1. バックエンドのコンパイル
Javaソースコードをコンパイルし、クラスファイルを生成します。

Bash
javac backend/sorter/*.java
2. アプリケーションの起動
Electronを起動し、GUIを通じてファイル仕分け機能を開始します。

Bash
npm start
